在非Honor/Samsung手机上使用微软的“连接至Windows”功能

发表于2023-04-20|更新于2024-11-21|技术分享
|字数总计:731|阅读时长:2分钟|阅读量:

前言

以下方法已过时。新方法在文末

尝试了各种手机+电脑的协同办公方案——小米系的MIUI+、Linux系的KDE Connect等。因为我的小米设备较多，所以MIUI+体验还不错。诸如屏幕镜像（扩展显示器）/剪贴板共享等功能都正常工作。唯一的小问题是只支持米系笔记本电脑那都不是事。最近想起来，微软自家有个“连接手机”的应用，打算尝试一下。不过貌似国区只支持部分荣耀手机、部分三星手机还有不存在的Microsoft Duo。不信邪，试试看！

在网上搜索了一番，最终在 非特定手机如何连接到Windows | heStudio 这里找到了答案。本文便以此为参照。

准备

  • 一台安卓(?)手机，要求系统大于7.0 可能苹果也行，没试过
  • 一台Windows10/11电脑 废话
  • 能正确上网的工具，节点任意
  • 耐心的态度 别急
  • 可选项：
    • 同一网络环境
    • WiFi6接入

开始！

手机 下载 国区特供版 连接电脑

笔者测试过了，Google Play商店下载的版本，电脑端会卡在“无法连接”的地方。老老实实用国区。

下载地址：Link To Windows

然后就是给一堆权限啥的 如果不想用电脑打电话看短信的话可以不给电话和短信权限

记得要给“管理所有通知”的权限

电脑 下载 连接手机

MS官方应用商店：手机连接 - Microsoft Store 应用程序

连接

打开PC端的“手机连接”，选择手机OS后会让你确保拥有受支持的设备有也不会看这个了

直接继续，在无代理的情况下登录。之后

因为Windows对UWP应用的限制，它们无法监听本地回环地址127.0.0.1，因此无法正常使用系统代理，并且在代理打开时无法正常上网。

可以通过工具来解除UWP应用的回环代理限制。

手机上扫码，出现下图就可以耐心等待了 转圈说明你网卡了

PC端出现下面说明连接好了

然后就可以享受丝滑的接受应用消息了😋

配置电话/短信

保险起见，在Windows蓝牙设备里先取消配对你的手机（手机上也和电脑取消配对）。

之后按着PC端的指示一步步在手机上给权限/配对

就可以用电脑唤醒手机打电话了😋

测试可行的新方法

2029.09.03 在Redmi K70 With HyperOS上测试成功

  1. 手机安装非国区版本的连接至Windows，并令其走代理
  2. 电脑安装手机连接，并开启代理
  3. 电脑端使用Microsoft账户登录
  4. 手机选择“扫描二维码进行连接”
  5. 耐心等待即可
